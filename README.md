# AI Playwright Learning — API Test Framework

Playwright + TypeScript API test framework.

## Prerequisites

- Node.js v18+
- Running test server: https://github.com/yungwenpeng/nodejs-example (`http://localhost:3000`)

## Setup

```bash
npm install
npx playwright install
```

## Run Tests

```bash
npm run test:api          # run all API tests
npm run test:api:report   # open HTML report
```
